# 1. Тест-кейсы: Оплата банковской картой

## Описание задачи

Веб-приложение позволяет пользователю оплатить заказ банковской картой через внешний платёжный шлюз.

Процесс оплаты включает:

- Нажатие кнопки «Оплатить»
- Клиентскую валидацию
- Отправку данных в платёжный шлюз
- Возможный редирект на 3-D Secure
- Получение одного из результатов:
  - Успешно
  - Отклонено банком
  - Ошибка платёжного сервиса
- Обновление статуса заказа
- Отображение результата пользователю
- Отправку email-чека (при успешной оплате)

Решение приведено в папке test_cases
